This application is designed to help tourists.

The user must first register.

The user can remove objects from the map by selecting the appropriate menu items.

This app is also designed to detect weather at a specific geographic location or in a specific city.

To find out the weather in some city by input enter the name of this city into the input with placeholder 'City Name' and click button "Search city".

To find out the weather at a particular point of the map, you must right-click the mouse and select the menu item "Create Marker".

If the marker has the form of a "cross" then it means that it is possible to obtain weather data from a settlement (city, town or village) by this coordinate.

If the marker has the form of a "hand", it is possible to obtain data about the weather in the nearest town, the data of which is in the bases of "OpenWeatherMap".

To find out the weather in a specific place where the marker is already installed, just click on the marker with the left mouse button.

Before creating a marker, you can select its color by using the menu item "Marker's color" and determine whether or not it is possible to drag this marker using the menu item "Marker's draggable".

At any time you can determine the possibility of rotation of the map after right-clicking the mouse using the menu item "Maps rotation".

At any time you can also define zooming and language of the map by using the appropriate menu item.

Please note that after changing the language on the map, you will not be able to create and delete the markers with the right mouse button.

You can remove the marker by right-clicking the mouse and selecting the item "Delete marker".

If the city name is different from the one you expect, you can remove the marker, zoom in on the map, and create a new marker in the appropriate location. You can also drag the marker to the corresponding point and select the menu item "Create popup".
